# Model Architectures

Architectures adapt prepared feature frames into train, predict, and evaluate operations. BEE currently supports binary scikit-learn classifiers.

## Shared behavior

- Numeric feature columns are selected automatically.
- `timestamp`, `target`, and `_row_id` are excluded from features.
- Training rows with null or non-finite float values are dropped.
- Prediction input nulls, NaN, positive infinity, and negative infinity are converted to `0.0`.
- If training labels contain fewer than two classes, the architecture falls back to `DummyClassifier(strategy=most_frequent)`.
- Evaluation returns `accuracy`, `precision`, `recall`, and `rows`.

## `logistic_regressor_arc`

Display name: **Logistic Regressor**. Uses scikit-learn `LogisticRegression` and supports probabilities through `predict_proba`.

| Hyperparameter | Default | Constraints | Meaning |
| --- | --- | --- | --- |
| `C` | `1.0` | number, 0.0001 to 1000.0 | Inverse regularization strength; smaller means stronger regularization. |
| `max_iter` | `200` | integer, 50 to 5000 | Maximum solver iterations. |
| `penalty` | `l2` | `l2` or `None` | Regularization penalty. Runtime maps empty/None to `l2`. |
| `solver` | `lbfgs` | `lbfgs`, `liblinear`, `newton-cg`, `sag`, `saga` | Optimization solver. |
| `class_weight` | `None` | `None` or `balanced` | Optional class imbalance weighting. |
| `fit_intercept` | `True` | boolean | Whether to fit an intercept term. |
| `tol` | `0.0001` | number, min 0.0 | Solver convergence tolerance. |

Prediction output shape:

- `_preds`: `list[int]` containing predicted class labels.
- `_probs`: `list[float]` containing class-1 probabilities when available.

## `ridge_classifier_arc`

Display name: **Ridge Classifier**. Uses scikit-learn `RidgeClassifier` and can produce probabilities only when calibration succeeds.

| Hyperparameter | Default | Constraints | Meaning |
| --- | --- | --- | --- |
| `alpha` | `1.0` | number, 0.0001 to 1000.0 | L2 regularization strength. |
| `fit_intercept` | `True` | boolean | Whether to fit an intercept term. |
| `class_weight` | `None` | `None` or `balanced` | Optional class imbalance weighting. |
| `tol` | `0.0001` | number, min 0.0 | Solver tolerance. |
| `calibrator_method` | `sigmoid` | `sigmoid` or `isotonic` | Calibration method for probability estimates. |
| `calibrator_cv` | `0` | integer, 0 to 10 | Cross-validation folds for calibration. Calibration only runs when this is at least 2 and enough labels exist. |

Prediction output shape:

- `_preds`: `list[int]` containing predicted class labels.
- `_probs`: `list[float]` only when calibration succeeds.

## Compatibility alias

`logreg_binary` is accepted as an older alias for `logistic_regressor_arc`, but metadata lists only canonical architectures.

## Metrics generated by architectures

| Metric | Meaning |
| --- | --- |
| `accuracy` | Fraction of labels predicted correctly. |
| `precision` | Fraction of positive predictions that were correct. Zero when undefined. |
| `recall` | Fraction of actual positives found. Zero when undefined. |
| `rows` | Number of cleaned test rows used for evaluation. |
