# Public Hub

The Public Hub helps authenticated users discover reusable research artifacts.

Visible records must pass these rules:

- Users must be enabled.
- Blueprints must be approved.
- Experiments must be completed and successful.
- Models must belong to completed successful experiments tied to approved Blueprints.

The hub supports tabs for users, experiments, models, and Blueprints, plus simple search.
