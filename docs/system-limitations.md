# System Limitations

BEE is intentionally scoped.

- Market data is centered on BTCUSDT.
- Long experiment windows and large permutation counts can be slow.
- Failed or cancelled experiments may have partial logs but are not exportable.
- In-memory sessions are suitable for local development, not multi-worker production.
- Results are experimental and should not be treated as trading advice.

Use exports for reproducibility and keep the raw experiment configuration with any external analysis.
