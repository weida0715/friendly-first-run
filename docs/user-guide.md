# User Guide

Most users follow this path:

1. Register or sign in.
2. Browse approved Blueprints or create a draft Blueprint.
3. Create an experiment from an approved Blueprint.
4. Wait for the experiment to complete.
5. Review model rankings, metrics, charts, and logs.
6. Export completed experiment artifacts when needed.

Keep experiments small while iterating. Increase the date range and permutation count only after the Blueprint behaves as expected.
