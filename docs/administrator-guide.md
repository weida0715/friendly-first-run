# Administrator Guide

Administrators and moderators keep the system usable and safe.

## Responsibilities

- Manage user status and roles.
- Review submitted Blueprints.
- Monitor system queue and settings.
- Keep public artifacts limited to approved, successful, reusable records.

Moderators can handle normal user operations. Administrators can manage elevated roles and system-level settings.
