# Wireframes (Bootstrap)

This directory contains **high-level static wireframes** that mirror the current `src/` app pages with:

- HTML + CSS (+ tiny JS for nav active state)
- Bootstrap 5 via CDN
- No app/business logic
- No API integration

## Files

- `index.html` (Landing)
- `login.html`, `register.html`
- `dashboard.html`
- `experiments.html`, `experiment-detail.html`
- `models.html`
- `sfms.html`
- `public-hub.html`
- `docs.html`
- `profile.html`, `user-profile.html`
- `admin.html`
- `settings.html`
- `not-found.html`

## Usage

Open any file directly in browser, for example:

- `src/wireframes/index.html`
